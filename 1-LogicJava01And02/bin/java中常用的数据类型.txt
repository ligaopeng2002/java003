

bit 比特 01 
byte 字节  1byte = 8bit （-128~127）
kb   千字节   1kb = 1024byte
mb   兆字节   1mb = 1024kb
gb   吉字节   1gb = 1024mb
tb   太字节   1tb = 1024gb 

整数：
int （4字节， 4byte ， 默认值0 ，  大约2的32次幂-1）
byte （1字节，默0 ，  -128~127）
short（2字节，默0）
long（8字节 ，默认0L）

非整数：
float（4字节，32位精度，默认0.0f）
double（8字节，64位精度，默认0.0d）

布尔类型
boolean（true 或 false ， 只有2个值， 默认值false）

单字符
char（2个字节  ，默认值是空格  ）

String 字符串型：
String 跟上面8中基本数据类型是有区别的，它是一种引用类型，如果不初始化是没有默认值的。是null值
